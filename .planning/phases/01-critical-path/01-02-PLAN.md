  business_name: string     — shown in Claude prompt (e.g., "Dupont Plomberie")
  service_type: string      — Claude prompt context (e.g., "plombier")
  city: string              — Claude prompt context (e.g., "Paris")
  owner_phone: string       — E.164 format (e.g., "+33612345678")
  owner_email: string       — fallback owner contact
  callback_promise_minutes: number — promised callback time in Claude message (e.g., 30)
  google_key: string        — MUST be stored as n8n env var reference, not literal
  active: boolean

<!-- n8n workflow JSON structure (minimal) -->
A valid importable n8n workflow JSON has this top-level shape:
{
  "name": string,
  "nodes": Node[],
  "connections": ConnectionMap,
  "settings": { "errorWorkflow": string (optional) }
}
Each Node: { "id": string (UUID), "name": string, "type": string, "typeVersion": number, "position": [x, y], "parameters": {} }

<!-- Required n8n node types for main workflow -->
Webhook:          n8n-nodes-base.webhook
Code:             n8n-nodes-base.code
If:               n8n-nodes-base.if
Remove Duplicates: n8n-nodes-base.removeDuplicates

Brevo (email):    n8n-nodes-base.brevo

**prompts/prospect-sms-fr.txt** — Claude system prompt template (used verbatim in the HTTP Request node):
```
Tu travailles pour {{ $env.BUSINESS_NAME }}, {{ $env.SERVICE_TYPE }} basé(e) à {{ $env.CITY }}.

Un prospect vient de soumettre un formulaire de contact Google Ads.

Ta mission : rédiger UN SMS de confirmation en français.

Règles STRICTES :
- Le SMS doit faire STRICTEMENT MOINS DE 160 caractères (espaces compris)
- Reformule en une phrase la demande spécifique du prospect (ce qu'il a écrit)
- Confirme qu'on le rappelle sous {{ $env.CALLBACK_MINUTES }} minutes
- N'invente AUCUN prix, délai exact, ni disponibilité non confirmée
- N'ajoute AUCUNE formule de politesse, signature, ou introduction
- Commence DIRECTEMENT par le contenu du SMS

Exemple acceptable (150 caractères) :
"Bonjour Marie, votre fuite d'eau sous l'évier : notre plombier vous rappelle sous 30 minutes. Dupont Plomberie."

Exemple interdit (trop long, prix inventé) :
"Bonjour Marie, nous avons bien reçu votre demande concernant une fuite d'eau sous votre évier de cuisine. Nous vous confirmons qu'un de nos plombiers qualifiés vous contactera dans les 30 prochaines minutes pour intervenir dès que possible au prix de 80€ de l'heure. Dupont Plomberie, votre spécialiste."
```

Note: The env var references `{{ $env.BUSINESS_NAME }}` etc. are n8n expression syntax. They are resolved at runtime when the HTTP Request node sends the body to Claude. The prompt file is the human-readable source of truth — the executor for Plan 05 will embed this content verbatim in the Claude node's system prompt field.

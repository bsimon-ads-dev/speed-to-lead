"parameters": {
  "jsCode": "// NOTIF-04: Error fallback — send raw lead to Baptiste when pipeline fails\n// Read lead data stored by main workflow's raw log node\nconst staticData = $getWorkflowStaticData('global');\nconst rawLead = staticData.lastLead ? JSON.parse(staticData.lastLead) : null;\n\n// Parse error context from Error Trigger\nconst workflowName = $json.workflow?.name || 'Speed to Lead';\nconst errorMsg = $json.execution?.error?.message || 'Erreur inconnue';\nconst lastNode = $json.execution?.lastNodeExecuted || 'noeud inconnu';\n\n// Build fallback SMS (keep under 320 chars — 2 SMS segments max)\nlet sms;\nif (rawLead) {\n  // Extract basic lead info for Baptiste\n  const fields = {};\n  for (const col of rawLead.user_column_data || []) {\n    fields[col.column_name] = col.string_value;\n  }\n  const name = fields['FULL_NAME'] || 'Inconnu';\n  const phone = fields['PHONE_NUMBER'] || 'pas de tél';\n  const request = fields['demande'] || fields['message'] || '';\n  const leadId = rawLead.lead_id || '?';\n  sms = `LEAD PERDU [${workflowName}]\\nErreur: ${errorMsg.substring(0, 60)}\\nLead: ${name} | ${phone}\\nDemande: ${request.substring(0, 60)}\\nID: ${leadId}`;\n} else {\n  sms = `LEAD PERDU [${workflowName}]\\nErreur: ${errorMsg.substring(0, 80)}\\nDernier noeud: ${lastNode}\\nAucune donnée lead disponible.`;\n}\n\n// Truncate total SMS to 320 chars (2 segments) as hard limit\nreturn [{ json: { fallback_sms: sms.substring(0, 320) } }];"
}
```

  "authentication": "genericCredentialType",

  "bodyParameters": {
    "parameters": [
      {
        "name": "To",
        "value": "={{ $env.BAPTISTE_PHONE }}"
      },
      {
        "name": "From",
        "value": "={{ $env.TWILIO_SENDER_ID }}"

        "value": "={{ $json.fallback_sms }}"
      }
    ]
  },
  "options": {}
}
```

Note on authentication: The `genericCredentialType` with `httpBasicAuth` causes n8n to prompt for a credential when importing. The executor will create an n8n Basic Auth credential with:
- Username: the value of TWILIO_ACCOUNT_SID env var
- Password: the value of TWILIO_AUTH_TOKEN env var

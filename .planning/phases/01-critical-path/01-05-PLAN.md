```json
"parameters": {
  "method": "POST",
  "url": "=https://api.twilio.com/2010-04-01/Accounts/{{ $env.TWILIO_ACCOUNT_SID }}/Messages.json",
  "authentication": "genericCredentialType",
  "genericAuthType": "httpBasicAuth",
  "sendBody": true,
  "specifyBody": "keypair",
  "bodyParameters": {
    "parameters": [
      {
        "name": "To",
        "value": "={{ $json.phone }}"
      },
      {
        "name": "From",
        "value": "={{ $env.TWILIO_SENDER_ID }}"
      },
      {
        "name": "Body",
        "value": "={{ $json.claude_message }}"
      }
    ]
  },
  "options": {}
}
```
CRITICAL: This uses HTTP Request node (not the built-in Twilio node) because the built-in Twilio node cannot set an alphanumeric sender ID — it only accepts E.164 numbers. France A2P requires alphanumeric sender registration.
